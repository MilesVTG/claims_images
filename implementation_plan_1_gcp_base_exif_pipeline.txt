**main.py**
```python
import json
import os
from datetime import datetime
from PIL import Image, ExifTags
from google.cloud import storage, bigquery

def extract_exif(event, context):
    """Triggered by new photo upload to GCS."""
    bucket_name = event['bucket']
    file_name = event['name']
    uri = f"gs://{bucket_name}/{file_name}"

    # Download image bytes
    storage_client = storage.Client()
    bucket = storage_client.bucket(bucket_name)
    blob = bucket.blob(file_name)
    image_bytes = blob.download_as_bytes()

    # Extract EXIF
    img = Image.open(io.BytesIO(image_bytes))  # need import io
    exif = img.getexif()
    metadata = {}

    if exif:
        for tag_id, value in exif.items():
            tag = ExifTags.TAGS.get(tag_id, tag_id)
            if tag == 'GPSInfo':
                # Parse GPS (standard helper function below)
                gps = parse_gps(value)
                metadata['gps_lat'] = gps['lat']
                metadata['gps_lon'] = gps['lon']
                metadata['location'] = f"{gps['lat']},{gps['lon']}"
            elif isinstance(value, bytes):
                metadata[tag] = value.decode(errors='ignore')
            else:
                metadata[tag] = str(value)

    # Insert into BigQuery
    bq_client = bigquery.Client()
    table_id = "fraud_detection.claims"  # or staging table
    rows_to_insert = [{
        "contract_id": extract_contract_from_filename(file_name),  # your logic
        "claim_id": file_name.split('/')[0],
        "photo_uris": [uri],
        "extracted_metadata": json.dumps(metadata),
        "processed_at": datetime.utcnow().isoformat()
    }]

    errors = bq_client.insert_rows_json(table_id, rows_to_insert)
    if errors:
        print(f"BigQuery errors: {errors}")
```

**parse_gps helper** (add to main.py):
```python
def parse_gps(gps_info):
    def dms_to_dd(dms, ref):
        degrees = dms[0][0] / dms[0][1]
        minutes = dms[1][0] / dms[1][1]
        seconds = dms[2][0] / dms[2][1]
        dd = degrees + minutes/60 + seconds/3600
        return dd if ref in 'NE' else -dd
    lat = dms_to_dd(gps_info[2], gps_info[1])
    lon = dms_to_dd(gps_info[4], gps_info[3])
    return {"lat": lat, "lon": lon}
```

```text

{list previous photo descriptions + tire brand + vehicle color + dates}

Current claim:
Reported loss date: {date}
Service drive: {location} at coords {coords}

Photos to compare (newest first):
[Image 1: current photo]
[Image 2..N: previous photos for this contract]

Tasks:
1. Extract visible tire brand(s) from all photos.
2. Extract vehicle color from all photos.
3. Check for recycled/dupe images (identical damage, lighting, angle).
4. Flag any inconsistencies: tire brand change, color change, damage healing vs fresh, timestamp/geo mismatch.
5. Overall fraud risk (0-100).

Return ONLY valid JSON:
{
  "risk_score": 85,
  "red_flags": ["Tire brand changed: Michelin → Goodyear", "Vehicle color: blue → red"],
  "explanation": "...",
  "recommendation": "Manual review + reverse image search"
}
```

**How to run at scale:**
- Option A (easiest for 100k+): Create BigQuery Object Table → `SELECT uri, AI.GENERATE_TEXT(...)` with prompt joining historical data.
- Option B (highest throughput): Generate JSONL in GCS → Submit Vertex AI Batch Inference job (Gemini 2.5 Flash or Pro) → output lands in BigQuery.
