### 1. Dockerfile (Single image for everything)

```dockerfile
FROM python:3.12-slim

WORKDIR /app

# System deps
RUN apt-get update && apt-get install -y \
    libgl1-mesa-glx libglib2.0-0 \
    && rm -rf /var/lib/apt/lists/*

COPY requirements.txt .
RUN pip install --no-cache-dir -r requirements.txt

COPY . .

# Run as FastAPI webhook (default) or polling mode
CMD ["uvicorn", "main:app", "--host", "0.0.0.0", "--port", "8080"]
```

### 4. Portable Terraform (Multi-Cloud Modules)

Use this structure:

```bash
terraform/
├── main.tf
├── modules/
│   ├── object-storage/     # AWS S3 or MinIO or GCS or Azure Blob
│   ├── database/           # RDS Postgres, Cloud SQL, Azure Postgres, or self-hosted
│   └── kubernetes/         # EKS, GKE, AKS, or generic K8s
```

Example snippet (AWS version – change provider for GCP/Azure):

```hcl
provider "aws" { region = "us-east-1" }

module "storage" {
  source = "./modules/object-storage"
  bucket_name = "your-photos-bucket"
}

module "postgres" {
  source = "./modules/database"
  db_name = "fraud_detection"
  instance_class = "db.t4g.medium"
}

module "k8s" {
  source = "./modules/kubernetes"
  app_image = "your-docker-image:tag"   # push to ECR / GAR / ACR
}
```

Switch providers in 5 minutes (aws → google → azurerm).
